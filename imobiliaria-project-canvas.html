01 / Visão do Produto

O que é o ImobiFlow?

Uma plataforma web para imobiliárias que centraliza leads, automatiza o atendimento via WhatsApp, gerencia o pipeline de imóveis e corretores — tudo integrado ao N8N como motor de automação.

Escalabilidade

Problema central: Imobiliárias perdem leads por atendimento lento, dados descentralizados entre WhatsApp/planilha/e-mail e falta de visibilidade do funil. O ImobiFlow resolve isso com um CRM visual + automação inteligente via N8N + atendimento 24h por WhatsApp com IA.

🎯
Público-Alvo
Imobiliárias pequenas e médias (2–30 corretores) que querem profissionalizar o atendimento sem grandes investimentos em tecnologia própria.
B2B SaaS Imobiliárias Corretores
💡
Proposta de Valor
Lead capturado → qualificado pela IA → atribuído ao corretor → acompanhado até o fechamento. Tudo automatizado, sem perder o contato humano.
Automação IA WhatsApp
📊
Modelo de Receita
SaaS recorrente por imobiliária (plano por corretores ativos) + setup fee + opcional: automações customizadas como serviço adicional.
Recorrente Setup Fee Upsell
Diferencial Competitivo

Por que não usar só um CRM pronto?

❌ Soluções genéricas
CRMs como Pipedrive e HubSpot não entendem o fluxo imobiliário (visitas, documentação, CRECI, portais como OLX/ZAP). Automações custam caro ou não existem para WhatsApp.
✅ ImobiFlow
Fluxo nativo para imobiliárias + automação real via N8N (sem black-box) + WhatsApp 24h com IA treinada no negócio + integrações com ZAP, OLX, VivaReal via webhook.
02 / Arquitetura de Módulos

Os 6 módulos da plataforma

Cada módulo é independente mas se comunica via API interna e aciona fluxos no N8N automaticamente.

🧲
M1 · Captação de Leads
Centraliza leads de múltiplas fontes: portais imobiliários, formulários no site, WhatsApp direto, Meta Ads e Google Ads. Cada lead entra no pipeline automaticamente.
ZAP Imóveis OLX VivaReal Meta Ads Webhook
🤖
M2 · Atendimento IA (WhatsApp)
Bot inteligente via Evolution API que qualifica o lead, entende a necessidade (compra/aluguel, bairro, faixa de preço), agenda visitas e transfere ao corretor no momento certo.
Evolution API Claude / GPT N8N AI Agent
📋
M3 · CRM & Pipeline
Kanban visual com estágios do funil imobiliário: Novo Lead → Qualificado → Visita Agendada → Proposta → Documentação → Fechado. Histórico completo de interações.
Kanban Timeline Notas Tarefas
🏠
M4 · Catálogo de Imóveis
Cadastro completo de imóveis com fotos, valores, características, localização. Motor de match automático: lead cadastrado → sistema sugere imóveis compatíveis.
Match IA Galeria Filtros Mapa
📅
M5 · Agenda & Visitas
Calendário de visitas integrado com WhatsApp. Confirmação automática (24h e 1h antes), link do Google Maps para o cliente, follow-up pós-visita automatizado.
Google Calendar Lembretes Follow-up
📈
M6 · Dashboard & Relatórios
Métricas em tempo real: leads por fonte, taxa de conversão por corretor, tempo médio de fechamento, imóveis mais visualizados. Relatório semanal automático por e-mail.
KPIs Gráficos E-mail Report

Arquitetura Multi-Tenant: Cada imobiliária tem seu próprio workspace isolado (schema PostgreSQL separado por tenant_id). Os fluxos N8N usam variáveis de ambiente por tenant. A Evolution API cria uma instância WhatsApp por imobiliária. Isso permite escalar como SaaS sem interferência entre clientes.

05 / Roadmap de Entrega

3 fases de desenvolvimento

Estratégia de entrega incremental: MVP funcional em 6 semanas, produto completo em 4 meses.

FASE 01
🚀 MVP — Core do Negócio
Semanas 1–6
  • Setup da infraestrutura (VPS, domínio, banco, N8N, Evolution API)
  • Autenticação multi-tenant (login por imobiliária)
  • Cadastro de corretores e imóveis (CRUD completo)
  • Pipeline CRM com Kanban (5 estágios fixos)
  • Fluxo N8N: recepção de lead via formulário + WhatsApp
  • Fluxo N8N: atendimento básico via IA + transferência ao corretor
  • Dashboard com métricas essenciais (leads, corretores, conversão)
FASE 02
⚙️ Automações Avançadas + Match IA
Semanas 7–12
  • Motor de match IA (lead → imóveis compatíveis ranqueados)
  • Agenda de visitas com Google Calendar integrado
  • Fluxos de lembrete (24h e 1h antes) e follow-up pós-visita
  • NPS automatizado via WhatsApp
  • Reengajamento de leads frios (régua automática)
  • Integração com portais: ZAP Imóveis e OLX (webhook)
  • Relatório semanal por e-mail (gerado pelo N8N)
FASE 03
📈 Escala SaaS + Funcionalidades Premium
Semanas 13–18
  • Painel de onboarding self-service (imobiliária se cadastra sozinha)
  • Planos de assinatura com Stripe (básico / profissional / enterprise)
  • App mobile (PWA ou React Native) para corretores em campo
  • Assinatura digital de contratos (DocuSign / ZapSign)
  • Módulo financeiro básico (comissões por corretor)
  • White-label: imobiliária usa o sistema com sua própria marca
  • Marketplace de automações N8N customizáveis
